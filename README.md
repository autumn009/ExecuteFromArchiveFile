# ExecuteFromArchiveFile
Execute from Archive file in MS-DOS

## 概要
　LHarcのアーカイブファイルの中にあるファイルを実行するためのMS-DOSコマンドです。パソコン新時代に流行ったDIETの原始的な実装のようなものです。


E コマンド


　という書式で使用します。

　詳しい使い方はver0.2(C)\E.MANをご覧下さい。

　実用性はありませんが、歴史的なソースとして公開するものです。


## ディレクトリ
ver 0.2(C) は、C版のソースコードです。

ver 0.2binary は、C版のバイナリーです。

ver 0.3(ASM) は、アセンブラ版のソースコードです。
